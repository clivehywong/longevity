# Getting Started

Use this section before running any workflow.

## Pages

- [`system-setup.md`](system-setup.md): environment requirements, install steps, and quick verification
- [`repository-orientation.md`](repository-orientation.md): where the data, scripts, results, and app live

## Recommended first-run checklist

1. Confirm the repository paths and datasets match your machine.
2. Install app dependencies if you plan to use `neuconn_app/`.
3. Validate BIDS naming before any preprocessing.
4. Read the workflow page for the task you actually want to run.
